BaseBall HOF Player Compiler

the application takes a year and displays all of the Hall of fame baseball players who played during that year. Can be used when opening baseball packs to know which baseball players to target from past years...

Install: install with python using vscode virtual environment and  "pip install e ."

Usage: After installing, in the terminal input "hof" then a space and input the year to receive a list of Hof players from that year. After 'hof'you can type 'print' then type a year, you will be asked to input your windows username and after the app will create an excel spreadsheet with the list of players from your search. 

Features: to receive a list of negro league players input negro either after 'hof' or after 'print' to receive an excel file. to receive a list of pre 1920's players input 'old' after 'hof' or after 'print' to create an excel 'file'. finally after 'print' type 'all' to receieve an excel file of all hof members...
